Detailed Description

The RKwardView class provides the view widget for the RKwardApp instance. The View instance inherits QWidget as a base class and represents the view object of a KTMainWindow. As RKwardView is part of the docuement-view model, it needs a reference to the document object connected with it by the RKwardApp class to manipulate and display the document structure provided by the RKwardDoc class.
